Tutorial Residence

Para un tutorial básico, ver el vídeo. Sí ya tienes conocimientos básicos de su funcionamiento, leé los comandos de despues del vídeo.



Comandos de selección.


/res select[y] [z]

Selecciona los cubos mediante X, Y y Z. El metodo habitual es el hacha de madera, pulsar click izquierdo en una esquina y click derecho en la otra.

/res select size

Muestra el coste y el tamaño de la res seleccionada.

/res select vert

Incluye toda la vertical a lo anterioremente seleccionado, desde la posicion 0 hasta la 255.


Comandos de información

/res info (residencia name)

Muestra toda la información de la res nombrada.

Ej

/res list (name)

Muestra la información de tus residencias o del nombre indicado en el campo opcional.

Ej

/res listall

Muestra todas las residencias del servidor.


Comandos de Utilidad

/res message (residencia) [enter/leave] [message]

Pone un mensaje de entrada/salida a la res nombrada, se puede omitir el nombre de la res si estas dentro de esta.

Ej

/res message (residencia) remove [enter/leave]

Elimina el mensaje de entra o salida de tú residencia.

/res rename [OldName] [NewName]

Cambia el nombre de tú residencia. Ej: /res rename spawn alarar

/res renamearea [residencia] [OldName] [NewName]

Cambia el nombre de un área de tu residencia. A veces puede dar problemas, no abusar mucho.


Comandos de creación

/res create [ResidenceName]

Crea una residencia, si has seleccionado dos puntos anteriormente, sí tienes dinero y si no hay residencias en medio.

Ej

/res area (add/remove) (residence) (areaID)

Agrega/elimina una zona a una res.

/res remove [ResidenceName]

Elimina una residencia.

Comandos de Flags

/res set (ResidenceName) [flag] [true/false/remove]

Da o quita permisos de las flags globales de una residencia.

Ej

/res gset (ResidenceName) [GroupName] [flag] [true/false/remove]

Le da/quita permisos a un grupo.

Ej

/res pset (ResidenceName) [PlayerName] [flag] [true/false/remove]

Le da/quita permisos a una persona en concreto.

Ej

/res mirror [resaraiz] [resdestino]

Copia todos los permisos de una res a otra.


FLAGS

Aquí están las flags que se utilizan en los comandos explicadas:

move - true deja libre movimiento

build - true deja libre construcción

use - deja libre uso de botones, placas, palancas etc.

container - deja usar baules, hornos, dispensores etc.

pvp - false no permite el PVP

ignite - false no permite el fuego

subzone - true permite crear subzonas

firespread - false no permite la propagación del fuego

damage - false no permite el daño fisico

tnt / creeper - false no permite el TNT ni la explosión de creepers

monsters - false no permite que spawneen mobs

flow - true deja fluir liquidos

tp - false no deja usar la teleportación


Comandos de renta

/res subzone [ResidenceName] [SubZoneName]

Crea una subzona dentro de una residencia de tu propiedad.

Ej

/res market rentable [Main residence name].[subplot name] [rent cost] [# of days] [repeat: true/false]

Alquila una res a el precio que tu decidas durante el tiempo que decidas cada cobro y da la posibilidad a volver a realizar o no el alquiler.

/res market [buy/info] [residence]

Compra o te informa de una residencia en venta.

/res market sell [residence] [amount]

Pone una residencia en venta al precio que eliges.

/res market unsell [residence]

Elimina una residencia de su estado de venta a normal.

/res lease [renew/cost] [residence]

Renueva o muestra el coste de una residencia.

/res market list

Muestra la lista de todas las residencias en venta.
