Dragon travel, ¡Volando con dragones!

Sí, habeis leído bien, a partir de ahora en cuanto el server esté operativo, se pondrá el Dragon travel, un plugin para crear rutas de vuelo entre distintos puntos del mapa.

Es otra forma más de viajar, y no le quita el mérito a nada. Pues eso, aquí teneis un pequeño fragmento de lo que será, ¡Saludos Mineplayers!.

Cuando el server esté abierto

Cuando abramos el server de nuevo, lo anunciaré por twitter, foro, página web, facebook, gritando en mi casa, ¡Preparaos porque vamos muy fuertes, algunas de las novedades!:

Se eliminarán los tps, y se implementará el metro, sortals y Dragon travel, además del /f home. No habrán más medios de transporte de momento

Los primeros días existirán más ofertas en relación a objetos, serán más económicos. También podreis encontrar alguna gana en subastas a horas aleatorias.
